About eckWMS

eckWMS is a warehouse management system developed by Bertruger Sp. z o.o. It's designed specifically for the needs of e-commerce businesses and service centers, focusing on efficient RMA processing, device tracking, and inventory management using Android clients, particularly on Movfast devices.

Key Features

  • Streamlined RMA (Return Merchandise Authorization) process management.
  • Device tracking using serial numbers and barcodes.
  • Basic inventory management for parts and devices.
  • Optimized for Android clients, especially Movfast hardware.
  • Multi-language support with dynamic translation capabilities.
  • Integration points for potential future extensions (e.g., Medusa shop).

Technology Stack

The system is built upon a modern and robust technology stack:

  • Backend: Node.js with Express.js framework.
  • Database: PostgreSQL managed via Sequelize ORM (migrating from a legacy file-based system).
  • Frontend: Modular vanilla JavaScript, HTML templates, CSS.
  • Authentication: Passport.js handling Local, Google OAuth2, and JWT strategies.
  • Translation: i18next library with OpenAI integration and server/client-side caching.
